## 파일을 이동시키거나 이름을 변경할때 사용.

### mkdir
- Make Directory
- mkdir (폴더명) 폴더생성

- mv (파일명) (바꾸고싶은 파일명) **파일 이름 변경**
- mv (파일명) (이동시킬 폴더명) 파일을 해당 폴더로 이동

- mv (패스/파일명) . 현재 작업중인 폴더의 위치로 해당 패스의 파일을 가져옴.
