## Page Object Pattern

어떠한 한 메뉴, 페이지의 모든 element들을 미리 찾아내놓고, `getter` 혹은 어떠한 행위를 묘사한 메소드를 이용하여 TestClass에서 element를 조작한다.

- Easy to Maintain
- Easy Readability of scripts
- Reduce or Eliminate duplicity
- Re-usability of code
- Reliability
